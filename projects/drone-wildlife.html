Drone-Based Wildlife Detection

UAV imagery · Object detection · Ecological monitoring

Overview

This project explores the use of UAV imagery for wildlife detection and density estimation in ecological surveys. High-resolution orthomosaics were processed to identify animal presence and spatial patterns.

Methods

  • UAV data collection and orthomosaic generation
  • Image preprocessing and tiling
  • Object detection workflow
  • Spatial density estimation

Results

The workflow successfully identified individual animals across large survey areas, demonstrating the feasibility of UAV-based monitoring for ecological research.
